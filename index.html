História dos Computadores

A história dos computadores começou com máquinas enormes e lentas, criadas para cálculos matemáticos complexos. Com o passar das décadas, os computadores evoluíram para dispositivos menores, rápidos e poderosos. Desde o ENIAC até os smartphones atuais, cada geração trouxe avanços tecnológicos que mudaram a sociedade. Hoje, usamos computadores em praticamente tudo, desde AI até sistemas embarcados.

Estrutura Básica do HTML

Todo documento HTML começa com uma estrutura fundamental que indica ao navegador como o conteúdo deve ser interpretado. Essa estrutura é formada por tags principais como <!DOCTYPE html>, <html>, <head> e <body>. Veja um exemplo:

<!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1>
  </body>
</html>

O <head> contém informações sobre a página (como título e metadados), enquanto o <body> contém o conteúdo visível, como textos, imagens e links.

Falando sobre Tags

As tags são os blocos fundamentais do HTML. Cada elemento é definido entre colchetes angulares como <tag> e </tag>. Existem tags para títulos (headings), parágrafos, imagens, links e muito mais. Algumas, como <br> e <hr>, não precisam de fechamento.

As tags são a linguagem que dá forma à web.

Atributos das Tags

Os atributos servem para definir propriedades extras de uma tag, como cor, tamanho, link e muito mais. Por exemplo, o atributo href é usado em links, e o atributo src em imagens. Exemplo:

<a href="https://www.dio.me" target="_blank">Visite a DIO</a>

Hoje, muitos atributos antigos, como os usados na tag <font>, foram substituídos por estilos CSS.

Textos

No HTML, podemos usar várias tags para formatar textos: negrito, itálico, sublinhado, marcado, riscado e texto menor. Também podemos representar fórmulas: H2O e 102 = 100.

Links

Os links conectam páginas e recursos na web. São criados com a tag <a> e o atributo href:

<a href="https://www.google.com" target="_blank">Abrir Google</a>

O atributo target="_blank" faz o link abrir em uma nova aba.
